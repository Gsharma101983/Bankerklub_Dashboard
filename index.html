🔗 Connect Your Google Sheet

One-time setup to sync your Master Dashboard live data. Takes about 3 minutes.

1
Open Apps Script in your Google Sheet

Go to your Master Dashboard sheet → click Extensions → Apps Script → delete existing code and paste the Code.gs provided in the companion file.

2
Deploy as Web App

Click Deploy → New Deployment → Type: Web app → Execute as: Me → Who has access: Anyone → Click Deploy → copy the URL.

3
Paste the Web App URL below

The URL looks like: https://script.google.com/macros/s/AKfy.../exec
